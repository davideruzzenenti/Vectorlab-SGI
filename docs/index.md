Documentazione ISO convertita da Word/Excel a Markdown e consolidata su GitHub.

## Struttura

I documenti sono raggruppati per area/clausola (niente più sottocartelle di stato working/da approvare/ecc. — lo stato e la cronologia sono gestiti tramite Git).

## Nota sulla consolidazione Master/Pubblicato

L'archivio originale su Google Drive conteneva due alberi paralleli, "Master" e "Pubblicato".
Verifica effettuata il 2026-07-09: i 9 file presenti in "Pubblicato" (tutti sotto
`7.5_INFORMAZIONI DOCUMENTATE`) erano copie identiche o più vecchie degli stessi file
già presenti in "Master" (in un caso, es. `MDPR08.1.D - formato EXCEL.xlsx`, la versione
Pubblicato risaliva al 2023-11-28 - 19.9KB, contro il 2026-04-30 - 83.4KB del Master).

Decisione: "Pubblicato" è stato scartato e "Master" è stato assunto come unica fonte di verità.
Nessun contenuto univoco è andato perso (verifica file per file eseguita prima dello scarto).
